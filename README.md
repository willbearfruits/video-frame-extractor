# Video Frame Extractor for Photogrammetry

A simple, locally running tool to extract high-quality frames from video files, optimized for photogrammetry workflows.

## Features
- **Smart Extraction:** Extract by time interval (e.g., every 1s), total count (e.g., 100 images), or frame step.
- **Blur Detection:** Automatically discard blurry frames to ensure high-quality reconstruction.
- **Privacy First:** Runs entirely on your computer. No data is uploaded to the cloud.

## How to Run
1. Double-click `install_and_run.bat`.
2. Wait for the installation to finish (first time only).
3. The application will open in your default web browser.

## Requirements
- Windows 10/11
- Python 3.8+ (Make sure "Add to PATH" is checked during installation)
